About This Project

This is a simple demonstration of a Progressive Web App (PWA) built with vanilla JavaScript.

What is a PWA?

Progressive Web Apps are web applications that use modern web capabilities to deliver an app-like experience to users.

Key Features

  • Works offline using service workers
  • Installable on devices
  • Responsive design
  • Fast loading
